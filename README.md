# Bot de Telegram - CNE Colombia

Bot de Telegram para consultar resoluciones del Consejo Nacional Electoral de Colombia.

## Características

- Acceso a 5,976 resoluciones del CNE (2023-2025)
- Modelos: Gemini 3 Flash + Gemini 2.5 Flash (fallback)
- Cita las resoluciones consultadas en cada respuesta

## Variables de Entorno

```
TELEGRAM_TOKEN=tu_token_de_telegram
GOOGLE_API_KEY=tu_api_key_de_google
```

## Deploy en Railway

1. Conecta este repo a Railway
2. Configura las variables de entorno
3. Deploy automático
